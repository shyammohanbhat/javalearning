## Leetcode patterns

| If problem says…     | Use            |
| -------------------- | -------------- |
| subarray / substring | Sliding Window |
| range sum            | Prefix Sum     |
| next greater         | Stack          |
| pair in sorted array | Two Pointers   |
| cycle                | Fast–Slow      |
| min/max answer       | Binary Search  |
| all combinations     | Backtracking   |
| optimize with memory | DP             |
